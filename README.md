# dio-trilha-python-workoutapi

# Ambiente virtual 
```sh
pyenv virtualenv 3.11.4 workoutapi
```
```sh
pyenv activate workoutapi
```
 ou
```sh
source ~/.pyenv/versions/workoutapi/bin/activate
```
Subir servidor
```sh
uvicorn workout_api.main:app --reload
```


Primeiro acesso
```sh
pip install -r requirements.txt
```
